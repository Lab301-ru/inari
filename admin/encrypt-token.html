Шифрование GitHub-токена

Запустите эту страницу один раз локально (или просто откройте в браузере). Введите PAT и придумайте пароль для клиентки — получите строку, которую нужно вставить в admin/config.js.

Скопировано в буфер обмена
  1. Создайте PAT на github.com/settings/tokens с правом repo (Contents: read/write для репозитория inari).
  2. Вставьте PAT и пароль выше, нажмите «Зашифровать».
  3. Скопируйте полученную строку и вставьте её в admin/config.js вместо СЮДА_ВСТАВИТЬ_ЗАШИФРОВАННУЮ_СТРОКУ.
  4. Закоммитьте и запушьте admin/config.js.
  5. Передайте клиентке только пароль. Адрес админки: https://inari-atelier.ru/admin/
